> 边境角色形象来源于 笛子太太 的《小绿和小蓝》

### 运行

> 配置 config.json 文件，在 ```bot_id``` ```password``` 配置机器人的QQ号和密码，```listen``` 配置你需要监听的QQ群。
>
> 执行 ```npm run start``` 启动机器人，途中可能需要滑动验证和设备锁验证。
>
> 注意：请使用 ```node.js 12.16``` 或以上版本。
